# Build Process
This folder features photos of the build process which was conducted as follows:
1. Cut frame pieces from cherry boards using table saw and miter saw
2. Cut base plate using plasma cutter
3. Weld supporting braces to bottom of base plate to mitigate bending
4. Laser cut acrylic side panels
5. Mill brackets to hold motherboard
6. Assemble frame and hang motherboard
7. Lathe standoffs and plasma cut hardware trays for disc drives
8. Complete detailing of panels with cherry slats
9. Assemble and wire components in case
